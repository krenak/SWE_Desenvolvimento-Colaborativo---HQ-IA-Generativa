# SWE_Desenvolvimento-Colaborativo---HQ-IA-Generativa
Os alunos deverão se dividir em equipes (de 3 a 5 integrantes) e deverão criar uma história em quadrinhos utilizando uma ferramenta de IA generativa (como ChatGPT, Gemini, DALL-E ou outras). As equipes devem seguir os princípios do SCRUM e Kanban ao longo do processo, utilizando o GitHub para gerenciar o conteúdo do projeto.
