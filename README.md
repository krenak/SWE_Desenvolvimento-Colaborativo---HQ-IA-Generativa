# Desenvolvimento Colaborativo - História em Quadrinhos + IA Generativa

(Descrição da atividade) Os alunos deverão se dividir em equipes 
(de 3 a 5 integrantes) e deverão criar uma história em quadrinhos utilizando
uma ferramenta de IA generativa. As equipes devem seguir os princípios do SCRUM
e Kanban ao longo do processo, utilizando o GitHub para gerenciar o conteúdo
do projeto.

## Objetivos da atividade

- Aplicar metodologias ágeis (SCRUM e Kanban) em um projeto criativo.
- Utilizar ferramentas de IA generativa para auxiliar na criação de 
conteúdo e arte.
- Promover a prática de desenvolvimento iterativo e incremental com GitHub.

### Descrição do projeto
Desenvolvimento de HQ utilizando ferramenta de versionamento (git) e hospedagem em nuvem (GitHub), com o gerenciamento utilizando SCRUM e o produto final gerado via VLM.

### Objetivos
Apresentar uma história fictícia (sci-fi) por meio de história em quadrinhos
utilizando versionamento e VLM para criar as imagens/quadrinhos
do tema proposto.

### Estrutura inicial da HQ
- Apresentação do contexto da HQ e dos personagens principais
- Estruturação do arco
- Desenvolvimento do arco
- Finalização do arco
